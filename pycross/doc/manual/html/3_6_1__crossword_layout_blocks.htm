3.6.1. Crossword layout blocks

 
When printed to paper or PDF, the crossword consists of several blocks as shown below:
 
Fig. 3.6.1.1. Crossword layout blocks
 

Crossword title

The crossword title taken from the meta (release) info.
See Editing the crossword information.
Top

Crossword release information

The meta (release) information about the crossword, like author, copyright, publishing date, editor etc.
See Editing the crossword information.
Top

Crossword grid

The crossword grid. Its appearance / style is configurable using Settings > User interface > Grid.
The Print settings let you include or exclude the grid from the printout and print it either with words or without.
Top

Crossword clues

The crossword clues. Clues are sorted by word number in the grid: first the Across words, then the Down words.
To the right of each clue, there is a hint with the number of letters (this hint can be turned off in the Print settings). You can also exclude Clues from the printout altogether.
If no clues are present (clues are not filled for the crossword), they will not appear in the printed crossword, without regard to the Print settings.
Top

Crossword clues title

The title for clues. The default is just "Clues", although this can also be configured in the Print settings.
