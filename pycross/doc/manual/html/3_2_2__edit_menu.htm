3.2.2. Edit menu

The Edit menu contains the following Main Actions:
 
Fig. 3.2.2. Edit menu actions

Undo

Undoes the latest operation (like modifying the crossword or clues, changing application settings, replacing the current crossword etc.)
Top

Redo

Redoes the latest previously undone operation.
Top

Edit

Toggles the Crossword grid editing mode.
Top

Clear

Clears all the word in the current crossword. See Editing words.
Top

Clear word

Clears the currently selected word in the grid (deleting only the non-bound characters in the word). See Editing words.
Top

Erase word

Erases the currently selected word in the grid (deleting all the characters in the word). See Editing words.
Top

Suggest word

Brings up suggestions for the selected word from the active word sources. See Peeking suggestions.
Top

Lookup word

Looks up the selected word's meaning in an online dictionary and/or Google. The found definition may be used straightforward as the word's clue. See Looking up word definitions.
Top

Edit clue

Switches the focus to the word's clue in the Clues panel.
Top

Clear clues

Clears all clues in the Clues panel.
Top

Multiselect

Toggles the Multiselect mode on or off to let you edit multiple cells at a time.
Top

Blank

Clears the selected cells in the grid. See Deleting letters.
Top

Block

Blocks the selected cells in the grid making them Blocked. See Blocking cells.
Top

Grey out

Makes the selected cells in the grid Surrounding Blocked. See Blocking cells.
Top

Add row

Adds a new row after the selected on in the crossword grid. See Adding and deleting rows and columns.
Top

Delete row

Deletes the selected row in the crossword grid. See Adding and deleting rows and columns.
Top

Add column

Adds a new column after the selected on in the crossword grid. See Adding and deleting rows and columns.
Top

Delete column

Deletes the selected column in the crossword grid. See Adding and deleting rows and columns.
Top

Duplicate

Duplicates the current crossword grid in any of the four directions, optionally reflecting and reversing the cells. See Duplicating the grid.
