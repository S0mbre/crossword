3.9.2. Adding and editing word sources

 
To manage the current word sources, go to Generate > Word sources or Options > Sources > Source management, or press the button in the Toolbar.
 

Source Manager

 
The Source Manager window will show up as shown in the figure below:
 
Fig. 3.9.2.1. Source Manager window

List of word sources ordered by precedence

List of added word sources. You can activate or deactivate a word source by checking / unchecking the checkbox in each word source. The source icon shows the type of word source:
  • -- database source
  • -- text file source
  • -- simple list source
 
The order of the word sources does matter! Higher-placed sources will have a higher precedence when generating crosswords, and vice-versa. For inactive (unchecked) sources, the order of precedence is ignored. You can use the Up and Down buttons in the Toolbar to move the sources and change their priority.
 
Double-clicking a word source brings up the Word Source Editor to let you change its parameters.

Source Manager toolbar

Use this Toolbar to manage your word sources. The Toolbar contains the following action buttons:
Move selected source up in the list (higher priority)
Move selected source down in the list (lower priority)
Add new source
Delete selected source
Edit selected source
Clear (delete) all sources
Launch the Word Database Manager
 
Top

Constrain maximum results checkbox

If checked, this option lets you constrain the number of matching words found during crossword generation. This option boosts generation performance when the active word sources aggregate to a very large dataset. When unchecked, no search result truncation will be made, and pycrossword will handle ALL the matches it finds.
Top

Maximum results threshold

Used in with the activated Constrain max results option, this box allows you to fine-tune the number of results yielded by the search engine (for each target word). The default value is 500.
Top
 

Word Source Editor

 
When you press the Add or Edit button in the Toolbar, the Word Source Editor window will show up:
 
Fig. 3.9.2.2. Word Source Editor window
 
The Editor lets you choose the name and type of the word source and configure its individual parameters.
 
You always begin by typing the name of the new / edited word source and selecting its type in the Source Type box.
