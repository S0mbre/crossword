3.7.4. Editing the crossword information

 
The crossword publication information, such as Author, Copyright etc., is stored with the crossword file (only in the IPUZ and XPF formats) and can be viewed and edited in pycrossword.
 
To view / edit the crossword information, go to View > Info, or press the button on the Toolbar, or use the Ctrl + I key combination. A tiny dialog will open up:
