3.7.2.2. Adding and deleting rows and columns

 
You can also add and remove whole rows and columns of the crossword grid, ultimately changing its dimensions. These operations are also performed with the Edit mode turned on.
 

Adding a new row

To add a new row, select the Edit > Add row menu or press the button on the Toolbar. The new row will be added before the current row (that with the active cell) or at the end of the grid if there is no active cell:
 
(a)
 
(b)
Fig. 3.7.2.2.1. A new row added (a) before the first row with the active cell, (b) at the bottom of the grid with no active cell
 
The new row has only blank non-blocked cells. So after adding it, you may want to split the new row into words with additional blocked cells as described in the previous section.
 

Deleting a row

To delete a row, select any cell in that row and then  select the Edit > Delete row menu or press the button on the Toolbar. Please note that you cannot delete a row unless it is selected, i.e. contains the active cell.
 

Adding a new column

In a similar way to adding rows, to add a new column, select the Edit > Add column menu or press the button on the Toolbar. The new column will be added before the current column (that with the active cell) or at the end of the grid if there is no active cell:
 
(a)
 
(b)
Fig. 3.7.2.2.2. A new column added (a) before the first column with the active cell, (b) at the right of the grid with no active cell
 
The new column has only blank non-blocked cells. So after adding it, you may want to split the new column into words with additional blocked cells as described in the previous section.
 

Deleting a column

To delete a column, select any cell in that row and then  select the Edit > Delete column menu or press the button on the Toolbar. Please note that you cannot delete a column unless it is selected, i.e. contains the active cell.
 
Since adding a removing rows and columns causes the application to reconstruct all words from scratch, all word clues will be lost after each grid modification! This behavior is likely to change in the future, but for now just remember that you will have to rewrite the clues once you're done with grid editing.
