3.9.2.3. List sources

 
You can create custom word lists directly in pycrossword by simply typing in or pasting words. Here's how it looks like:
 
Fig. 3.9.2.3.1. Configuring a list word source

Word source name

Type a name for this word source, e.g. "englist"
Top

Has parts of speech

Check this box if your custom list contains part-of-speech data (like in this example). Uncheck it if you won't need part-of-speech filtering.
Top

Part-of-speech delimiter

Type in or select a delimiter symbol for part-of-speech data (this example uses the slash symbol). The SPACE and TAB delimiters must be selected from the menu rather than typed in.
This drop-down box is only active when the Has parts of speech box is checked.
Top

Word list

This is where you put your words (optionally, with parts of speech). pycrossword will use these words (together with other word sources, if any) to generate crosswords.
Top

Shuffle

With this option activated (by default), the search results will be shuffled randomly during crossword generation. When unchecked, the results will always reflect the original ordering (as in the word list).
