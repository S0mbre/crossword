3.7.1.1. Peeking suggestions

 
pycrossword lets you look up and use suggestions for any incomplete word (having at least one blank letter). Suggested words are retrieved from the active Word sources.
 
To show suggestions for the selected word, choose Edit > Suggest word, or press the button on the Toolbar, or use the Ctrl + F hotkey.
 
The suggestions will be shown in a separate dialog:
 
Fig. 3.7.1.1.1. Word lookup (suggestions) dialog
 

Selected word

The selected word. Since it contains blanks, its pattern looks as follows: "_a_". Found suggestions match this pattern.
Top

List of suggestions

List of suggested words found in the active Word sources. See Working with word sources to learn about Word sources.
Top

Refresh suggestions

Press this button to reload suggestions from the Word sources.
Top

Sort suggestions

Sorts the suggestion list from A to Z.
Top

Shuffle suggestions

Shuffles the suggestion list in a random manner.
Top

Configure Word sources

Lets you configure (add / edit / remove / reorder / [de]activate) your Word sources. See Working with word sources to learn about Word sources. 
Top

Number of found suggestions

Displays the current number of found suggestions.
Top

Truncate suggestions

Check this box to truncate the found suggestions by the max threshold value. This value is set in Settings > Sources > Source management > Constrain max results.
When unchecked, no truncation will be applied and all the results will be displayed in the list.
Top
 
If you select a suggested word in the list and press the OK button at the bottom of the dialog (or double-click the suggested word), that suggestion will be put into the crossword grid replacing the selected word. If you press the Cancel button, no changes will be made to the crossword.
